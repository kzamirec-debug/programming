# Скрипти та Структури Даних Застосунку

У цій папці містяться вихідні файли, які описують та зберігають дані для рендерингу вебсторінок онлайн-магазину товарів для мистецтва.

### Опис архітектури даних:
Всі елементи з текстової таблиці (каталогу товарів) переведені у програмований тип даних Python безпосередньо у пам'яті. 
Основний файл скрипту: `art_data.ipynb`.

* `Категорія` - одна з 8 основних категорій товарів із прототипу (Фарби, Пензлі, Блокноти тощо).
* `Назва товару` - унікальне комерційне найменування художнього матеріалу.
* `Вартість` - ціна товару в гривнях (числовий формат для зручності математичних обрахунків у кошику).
* `Артикул` - унікальний текстовий ідентифікатор позиції (наприклад, `PT-001`, `BR-003`).
* `Статус наявності` - поточний стан складу («В наявності», «Закінчується», «Під замовлення»).
