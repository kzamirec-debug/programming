#  Прототипи Web-Застосунку

Ця папка містить графічні прототипи низької деталізації для інтернет-магазину художніх матеріалів та товарів для мистецтва.

### Вміст папки:
1. `ГоловнаСторінка.png` — **Головна сторінка**. Включає навігаційне меню, інтерактивну сітку для швидкого вибору категорій товарів та футер.
2. `ОпціяКаталогу.png` — **Сторінка Каталогу**. Демонструє структуру бокового динамічного меню ("Каталог", "Топ Продажів"), інтерактивні фільтри та основну сітку з товарів для вибору матеріалів.
